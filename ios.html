Setup Guide

  1. Download iKontroller from the iOS App Store
  2. Grant Local Network permission when prompted
  3. Open the app and enter your username
  4. Ensure your iPhone/iPad is on the same Wi-Fi as your Mac
  5. Wait for your Mac to appear in the device list
  6. Tap to pair and enter the 6-digit code shown on your Mac

Creating Custom Layouts

Step 1: Create a New Layout

  1. Tap the Layouts button in the top-left
  2. Tap the + button to create a new layout
  3. Enter a name for your layout (e. g., "FPS Game", "Minecraft")

Step 2: Add Controls

  • Buttons: Drag from the palette, resize, position, and tap to map to a key
  • Joysticks: Drag to position, map each direction (up, down, left, right)
  • Touchpad: Add for mouse movement and clicking
  • Scroll Wheel: For vertical/horizontal scrolling

Step 3: Map Keys

  1. Tap a control to open mapping options
  2. Tap "Map Key"
  3. Press any key on your Mac's keyboard
  4. The key will be captured and assigned
Note: Custom layouts require iKontroller Gold subscription. The default layout is free and supports up to 10 controls.

Connection Methods

Wi-Fi (Recommended)

  • Lowest latency when on 5GHz network
  • No cables required
  • Automatic reconnection

Peer-to-Peer (AWDL)

  • Works without a Wi-Fi router
  • Direct device-to-device connection
  • Slightly higher latency than Wi-Fi

USB Cable

  • Zero wireless latency
  • Most stable connection
  • Requires Lightning or USB-C cable

Troubleshooting iOS Issues

App Not Finding Mac

  • Check Local Network permission: Settings → iKontroller → Local Network
  • Ensure both devices are on same Wi-Fi network
  • Restart both apps
  • Check Wi-Fi settings (not using VPN or guest network)

Controls Not Responding

  • Ensure controls are mapped to keys
  • Check connection status indicator
  • Try disconnecting and reconnecting
  • Verify Mac Accessibility permissions are granted

Haptic Feedback Not Working

  • Check Haptic Feedback toggle in settings
  • Adjust haptic intensity
  • Note: Some older devices have limited haptic capabilities
